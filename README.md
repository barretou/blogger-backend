# 📘 API Project

A RESTful API built with **.NET 10**, focused on managing posts, authors, and authentication using JWT.

---

# 🚀 Features

- 📝 Posts management  
- 👤 Authors management  
- 🗂️ Categories support  
- 🔐 JWT Authentication & Authorization  
- 🛢️ Relational database with proper entity relationships  

---

# 🔐 Authentication

This API uses **JWT (JSON Web Token)** authentication.

## ✔️ Features
- Secure login endpoint
- Token-based authorization
- Protected routes
- Improved authentication security

---

# 🗃️ Database

The project uses a relational database with the following core entities:

- **Posts**
- **Authors**
- **Categories**

## ✔️ Highlights
- Proper foreign key relationships
- Fixed constraints between Posts and Authors
- Evolving schema through migrations
